# Issue 1: No Internet Connection

## Possible Causes

- Router connectivity issue
- Incorrect IP configuration
- Network adapter problem
- DNS issue

---

## Troubleshooting Steps

☐ Check physical connections

☐ Confirm Wi-Fi or Ethernet connection

☐ Restart network equipment

☐ Check IP configuration using ipconfig

☐ Test connectivity using ping

# Issue 3: Unable to Access Website

## Possible Causes

- DNS problems
- Browser cache
- Website availability issues

---

## Troubleshooting Steps

☐ Test another website

☐ Clear browser cache

☐ Test DNS resolution using nslookup

☐ Restart browser
